# Movies
一款电影介绍 APP， 客户端从 THE MOVIE DB 获取影视信息和剧照等数据。

请自行修改 buile.gradle 中 THE_MOVIE_DB_API_KEY
Please modify THE_MOVIE_DB_API_KEY at "app\build.gradle".
